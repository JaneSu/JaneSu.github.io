gulp安装和使用

正好今天比较闲，折腾一点新东西。目前就我需要的功能，不用gulp也能实现，但是gulp好像集成的功能更丰富一点。

那接下来我就述说安装的步骤和这个过程遇到的问题以及解决方法。

1、 全局安装gulp

Windows下使用以下命令：

npm install -g gulp

mac下使用以下命令：

sudo npm install -g gulp

2、 安装package.json文件

这里着重说明一下，不论是win还是mac都可以通过以下两个命令查看或是选择文件路径

cd，定位到目录
ls，列出文件列表

建议多敲敲这两个命令，了解文件系统并知道文件都在哪里。

3、 创建gulpfile.js文件

这一步有分歧，我看有的人操作方法是跳过了创建gulpfile.js文件，直接讲gulp安装到本地，在Windows系统上我没试过，但是用mac这么做的话就会一直报错，像下面这样：

4、 在当前的项目文件中安装gulp

必须在完成第三步之后在进行这里

npm install —-save-dev gulp

5、 安装需要的插件

npm install gulp-jshint gulp-sass gulp-concat gulp-uglify gulp-rename --save-dev

这是我在一个网站里看到的命令行，问题是执行这个命令之后，安装的插件会出问题

Error: Cannot find module ‘gulp-sass’ 
Error: Cannot find module ‘jshint/src/cli’

据说这是由于后期gulp升级导致的错误，所以还需要另外两段命令安装插件

npm install –save-dev jshint gulp-jshint 
npm install gulp-sass –save-dev

所以呢，我很怀疑上面安装插件那个命令是不是可以不用执行，但是我目前没有试过，所以以后做项目测试过了之后我再补充这一部分吧。

6、 使用

可以直接在webstorm中点击相应的事件，或是在命令行中输入gulp (task)。
