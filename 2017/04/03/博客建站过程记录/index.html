下一步添加添加SSH-Key

敲回车，会在根目录下的.ssh文件夹里生成两个文件，分别是id_rsa和id_rsa.pub.拷贝后者的内容，登录网页版 Github，点击头像 –> Settings –> SSH and GPG keys –> 点击 New SSH key 按钮 –> Title随意，将刚刚复制的内容粘贴到 Key 中 –> Add SSH Key 结束。

我用的osx，如果在根目录下找不到.ssh文件的话，打开终端，输入命令：

defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder

显示隐藏的文件内容。

操作完成后隐藏内容：

defaults write com.apple.finder AppleShowAllFiles -boolean false ; killall Finder

继续执行配置命令

到这里，网站基本就建成了，打开你设置的地址（http://用户名.github.io）就能看到由hexo生成的博客主页啦。

之后操作博客都需要通过命令行，任何操作过后，执行以下命令：

hexo d -g

另外一个常用到的命令是：

hexo new title "题目"

这个操作会在hexo/source/_posts下面生成一个新的md文件，这也是你页面显示文章放置的地方，修改这里，然后执行上面的命令，就能修改博客内容了。

七、主题

原生的博客主题，说真的，很丑。所以根据喜好可以选择一个自己喜欢的主题，我用的是next

通过cd命令进入/hexo/theme文件，运行命令下载主题：

git clone https://github.com/iissnan/hexo-theme-next.git

在安装插件

npm install hexo-renderer-jade --save
npm install hexo-renderer-sass --save

据其他教程提到的，这一步必须进行，否则会报错。

在theme文件夹内会生成一个新的文件夹，改名为next,同时修改_config.yml文件里的theme值为next。这样，主题就安装完成了。执行：

hexo d -g

刷新网页就能看到新安装的主题了。
