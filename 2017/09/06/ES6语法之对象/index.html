ES6语法之对象

发表于 2017-09-06 | 分类于 JavaScript |

正好用到一些es6中的对象语法，觉得很简洁，整理一篇笔记出来。

1.obj扩展方法

目前看到的扩展方法有两个：

  1. is方法
  2. assign方法

一个一个来说吧。

is方法

这里面我判断了一个数组和一个字符串是否相等，应该说，is方法在效果上相当于===所做的比较。

可能有些人会糊涂为什么两个一样的数组作比较，结果是false，关于这个我也是第一次发现，google了一下，有个解释是这样的：

因为数组是对象，引用语义，===不会比较两个数组的内容，而是比较两个引用是否相同。

我的理解是这样，举个例子，从一堆书里抽出来两本书，不同的出版社，同一个作者写的。尽管书名一样，但不能完全说这两本书是同一本书，因为出版社、排版、装帧什么的都不一样。

assign方法

Object.assign(obj1,obj2)有两个参数，分别是两个对象，该方法的作用是将obj2对象里的内容拷贝进obj1里面，就像使用电脑的时候，从一个文件夹把文件复制到另一个文件夹里。

结果会返回一个新的对象。

但假如，两个对象里有同样的一个key值怎么办？尝试一下。

结果很明显了，obj2里的键值对会覆盖obj1里面的。

另外需要强调一点，assign方式是浅拷贝。深浅拷贝的区别就自己查吧（因为我也不太清楚，叹气）。
