这里可以看一下，红色并不是一层，而是两层叠起来的，之所以这样，可以再加这一部分样式看一下：

如果删掉.right里的transform，你会发现这还是一个整园。这里的思路是将两个子div用clip切割，左半边是一个半圆，右半边也是一个半圆，所以看起来是一个圆，但事实是两部分组合而成的。再用动画旋转右边的半圆就可以实现饼图的效果了。

后期如果有更复杂的需求，可以配合js进行判断，自动调整旋转的角度，实现动态的数据展示。
