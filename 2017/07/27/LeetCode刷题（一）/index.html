LeetCode 344. Reverse String

题目

將一個字串反轉後回傳。
範例： s= “hello”, return “olleh”

思路

方法一：将字符串转换成数组，用遍历的方法从后向前查找元素。
方法二：利用js自身反转数组的方法reverse()改变数组的顺序。

LeetCode 242. Valid Anagram

题目

給兩個字串s與t，回傳t是否為s的重組字
範例：
s = “anagram”, t = “nagaram” 回傳true
s = “rat”, t = “car” 回傳false

思路

1.将字符串转换成数组
2.嵌套遍历两个数组的内容，如果两个相同，分别在该数组中删掉这个内容
3.每次判断相同重置数组的长度
4.如果剩下的最后一个长度的数组一致，则证明这两个字符串是同一个字符串
